## 修改內容說明

### 1. 後端 Python 腳本優化 (`fetch_mlb.py`)
- 在 `fetch_daily_transactions(date_str)` 解析邏輯中，新增對交易敘述（Description）的不區分大小寫過濾：
  ```python
  if "minor league" in desc.lower():
      continue
  ```
- 如此可以自動將與小聯盟深度合約（如 `"minor league contract"`、`"minor league roster status change"` 等）相關的資訊排除在戰報之外，避免大量無大聯盟出賽機會的邊緣球員簽約干擾每日戰報版面。
