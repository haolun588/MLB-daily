### 2. Python 抓取與生成核心實作
- **新建** [fetch_mlb.py](file:///c:/GitHub/MLB-daily/fetch_mlb.py)：
  - **高效 API 串接**：呼叫 MLB Stats API 時引入 `hydrate=linescore,decisions`，一次性取回賽程、局數、比分、球隊戰績及決定投手。
  - **自動球隊名稱翻譯**：建立大聯盟 30 支球隊之中文名稱、縮寫與官方代表色對照表，使生成之網頁卡片具備球隊客製化視覺邊框。
  - **戰績與比分精簡**：在球隊名稱旁動態加上當季戰績 `(勝-敗)`。比分板改以 R-H-E 格式呈現，並自動標記勝隊。
  - **打擊與投手焦點篩選**：實作 Notable Performers 篩選計分機制，精簡打者與投手名單，且全壘打/盜壘能自動帶入賽季累計數據格式如 `HR: 2(22)(23)`。
  - **開工排程檢查**：支援 `--check-wait` 參數，能於台北時間中午 12:00 定時執行並監控昨日賽事直至全數 Final 結束。
  - **首頁自動同步更新**：生成日戰報後，自動更新 `index.html` 的歷史存檔清單與最新一期焦點卡片。
