## 修改內容說明

### 1. 後端 Python 腳本優化 (`fetch_mlb.py`)
- 在 `fetch_daily_transactions(date_str)` 函數中，新增對含有 `"signed"` 關鍵字的交易事件進行大聯盟特徵比對：
  ```python
  if "signed" in desc_lower:
      indicators = ["free agent", "major league", "extension", "arbitration", "-year"]
      if not any(ind in desc_lower for ind in indicators):
          continue
  ```
- **核心目的**：
  - 過濾掉大量單純的選秀簽約（如 *"signed OF Eric Booth Jr."*）或其餘無大聯盟層級指標的普通簽約（如 *"signed RHP John Smith"*）。
  - 精準保留並顯示具備自由球員簽約（`"free agent"`）、大聯盟合約（`"major league"`）、延長合約（`"extension"`）、薪資仲裁（`"arbitration"`）或合約年限（`"-year"`，如 `"1-year"` 等）的大聯盟等級重要交易。
