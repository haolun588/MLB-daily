## 修改內容說明

### 1. HTML 模板新增球員連結樣式 (`templates/report_template.html`)
- 在 CSS 樣式區塊中新增 `.player-link` 的樣式設定：
  - 設定預設顏色為大聯盟主題藍色 `var(--accent-blue)`，粗體。
  - 新增平滑的顏色過渡動畫 `transition: color 0.2s ease`。
  - Hover 狀態下將文字改為亮藍色 `#60a5fa` 並加上底線，增加操作的直覺性與質感。

### 2. 投球決定連結化與 Bug 修復 (`fetch_mlb.py`)
- **連結化**：在「勝投」、「敗投」與「救援」三個欄位中，使用 `dec_obj["winner"]["id"]` 等 ID，將投手姓名改為超連結。
- **Bug 修復**：修正當無法對應至 boxscore ball-players 名單時（例如 `name` 為 `None`），原程式仍嘗試執行 `stats.get`（其中 `stats` 也是 `None`）而可能拋出 `AttributeError` 的 Bug，新增安全判斷排除此錯誤。

### 3. 單場焦點球員連結化 (`fetch_mlb.py`)
- 在打擊焦點 (Notable Batting Performers) 與投手焦點 (Notable Pitching Performers) 列表的生成邏輯中，將球員姓名包覆在超連結標籤中，導航至 `https://baseballsavant.mlb.com/savant-player/{playerId}`。

### 4. 人事異動描述文字姓名自動超連結化 (`fetch_mlb.py`)
- 在 `fetch_daily_transactions` 函數解析人事異動時，自 API 回傳的 transaction 物件中提取 `person` 的 `id` 與 `fullName`。
- 若該球員姓名確實存在於 `description` 當中，即動態以 `replace` 方式將姓名部分替換為對應的 Savant 超連結 HTML 字串，使比賽卡片內部的 Roster 異動及頁尾的「其他球隊人事異動」同步享有連結化功能。

### 5. 進度表更新
- 更新了 [progress.md](file:///c:/GitHub/MLB-daily/docs/progress.md)，將階段十標記為已完成。
