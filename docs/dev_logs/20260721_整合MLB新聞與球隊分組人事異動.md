## 修改內容說明

### 1. 樣式優化與模板調整 (`templates/report_template.html`)
- 在 `<style>` 中新增 `.game-news-box`、`.game-transactions-box`、`.card-team-tx`、`.card-tx-list` 及其子元素等樣式，採用柔和微光發光描邊效果。
- 修改 Game 1、Game 2 的靜態模擬卡片，在下半部左欄（投球決定下方）置入 Mock 新聞和卡片內異動框，提供使用者本地直觀預覽。
- 更新頁面底部的 `<!--TRANSACTIONS_SECTION-->` 佔位符為「其他球隊人事異動」設計。

### 2. 後端 Python 腳本串接 (`fetch_mlb.py`)
- **MLB 官方新聞摘要串接**：
  - 實作 `fetch_game_news(game_pk, ...)`，獲取官方新聞標題 (`headline`) 與簡介 (`blurb`)。
  - 設計防禦機制：若 API 尚無新聞（記者未撰寫完成），自動以本場完賽比分組成備用摘要，防範版面留白。
- **人事異動解析與動態分組匹配**：
  - 實作 `fetch_daily_transactions(date_str)` 獲取當天大聯盟（`sportId=1`）所有異動，並進行 `description` 去重。
  - 在 `generate_report(...)` 中，分析當日有出賽球隊，將其人事異動直接提取並合併至該場賽事卡片內展示；其餘無比賽球隊之交易，則彙整於網頁底部的「其他球隊人事異動」區塊。
- **無賽事日（Offseason）兼容**：
  - 支援在無任何賽事日（例如 2024-11-20）依然拉取大聯盟人事異動並在底部進行以隊帽分組顯示，防止 offseason 時出現空白網頁。
